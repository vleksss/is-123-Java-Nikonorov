# Auction

Веб-приложение аукциона на Java 17 и Spring Boot.

## Роли
- USER — участвует в аукционах и делает ставки
- OWNER — создаёт свои аукционы и просматривает ставки по ним
- ADMIN — управляет пользователями и всеми аукционами

## Возможности
- регистрация и авторизация
- каталог аукционов на Thymeleaf, Mustache и FreeMarker
- личный кабинет пользователя
- история ставок и список аукционов, где пользователь участвовал
- кабинет владельца с управлением своими аукционами
- панель администратора с блокировкой пользователей и изменением ролей
